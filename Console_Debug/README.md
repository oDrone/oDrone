Console debug
=============

Deploiement
-----------

1. Ajouter l'application sur le téléphone
Brancher le téléphone en mode developpeur
```
$ cd ClientDrone
$ ant debug install
```

2. Lancer le serveur
```
$ cd Server
$ node server.js
```

3. Ouvrir la console
Ouvrir le fichier ClientWeb/index.html dans un navigateur.
